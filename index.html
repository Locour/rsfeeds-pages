Why can't it just run make build command properly as I defined in Makefile? shyamajp@ubuntu:~/Projects/my-basic-app$ make build make: 'build' is up to date. Let's get rid of this stupid error. At the root level, you must have a directory named build. That's why you see this unexpected behavior. In my case, I can change outDir name from build to dist to avoid this specific conflict. tsconfig.json { "outDir": "dist" } Do not name your directory same as Makefile target. Be careful with names like build and test, which are often used for both Makefile targets and directory names.
